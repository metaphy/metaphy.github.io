**下面是v2ray的安装与配置：**

1. Ubuntu上安装v2ray
  - `curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh | sudo bash`

2. 启动，并且设置开机自动启动
  - `systemctl enable --now v2ray`

3. 配置如下(使用uuidgen获取uuid)
  - `vim /usr/local/etc/v2ray/config.json`
  - `内容如下` 

4. 防火墙开通端口
  - `sudo ufw allow 38388`
  - `sudo ufw status`

5. 本地profile配置：
  - `内容如下`
  
6. 本地连接Proxy Mode: Rule, Group: Manual-Selection, Proxy: ubuntu-vmess

7. 另外，服务器配置好后，做一个镜像：Server Instance -> Snapshots -> Take Snapshot


```
#服务器配置： /usr/local/etc/v2ray/config.json

{
    "log": {
        "loglevel": "warning"
    },
    "inbounds": [
        {
            "listen": "0.0.0.0",
            "port": 10086,
            "protocol": "vmess",
            "settings": {
                "clients": [
                    {
                        "id": "YOUR-GEN-UUID",
                        "alterId": 0
                    }
                ],
                "disableInsecureEncryption": false
            }
        }
    ],
    "outbounds": [
        {
            "protocol": "freedom"
        }
    ]
}


#本地Profile:  

proxies:
  - name: "ubuntu-vmess"
    type: vmess
    server: xx.xx.xxx.xx
    port: 10086
    uuid: 8xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
    alterId: 0
    cipher: auto
    udp: true


proxy-groups:
  - name: "Manual-Selection"
    type: select
    proxies:
      - "ubuntu-vmess"
      - DIRECT

rules:
  # 1. 常用国外网站走代理 (可选增加)
  - DOMAIN-KEYWORD,google,Manual-Selection
  - DOMAIN-KEYWORD,github,Manual-Selection
  - DOMAIN-KEYWORD,youtube,Manual-Selection

  # 2. 自动识别中国域名和 IP，强制直连
  - GEOSITE,cn,DIRECT
  - GEOIP,CN,DIRECT,no-resolve

  # 3. 兜底策略：剩下的流量全部走代理
  - MATCH,Manual-Selection
```

3. 记得查看防火墙状态，并把端口打开
  - `sudo ufw status`
  - `sudo ufw allow 38388`
  - `sudo ufw reload`

4. 本地Mac安装ShadowsocksX-NG.dmg，安装后提示不允许运行，使用以下命令打开
  - `sudo xattr -rd com.apple.quarantine /Applications/ShadowsocksX-NG.app`

5. 打开后配置Servers -> New Server，运行时选择PAC模式
  - `Auto Mod By PAC`

6. 如果远程server已经配置SSH Key，本地使用key登陆
  - `ssh -i ~/.ssh/idrsa root@70.1.1.1`
